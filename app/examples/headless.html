GraphSense - Headless Example

This is a headless example of GS running without any UI. Open the browser console (F12) and interact with the system:

Try These Commands


// View the system status
GS.core.status()

// Create an entity
GS.graph.addEntity({ id: 'repo1', type: 'repository', metadata: { title: 'My Repo' } })

// View all entities
GS.graph.serialize()

// View event history
GS.events.getHistory()
